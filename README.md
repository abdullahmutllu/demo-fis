# GiderTakip — Demo

İşletme gider takibi prototipi: fiş önizleme (simüle OCR), manuel gider, liste, özet ve analiz. Veriler tarayıcıda saklanır.

## Geliştirme

```bash
npm install
npm run dev
```

## Derleme

```bash
npm run build
```
